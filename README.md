# 🌱 Agro Forte | Futuro Sustentável

## 📖 Sobre o Projeto

**Agro Forte** é uma plataforma web educativa e interativa desenvolvida para promover práticas sustentáveis na agricultura e pecuária. O projeto apresenta informações sobre manejo ecológico, conservação dos solos, tecnologias agrícolas, bem-estar animal e ferramentas interativas para conscientização ambiental.

O objetivo é demonstrar como a produção agrícola pode coexistir de forma equilibrada com a preservação dos recursos naturais.

---

## 🚀 Funcionalidades

### 🌾 Práticas Sustentáveis

* Agricultura de Precisão
* Rotação e Consorciação de Culturas
* Manejo Integrado de Pragas (MIP)
* Sistemas Agroflorestais
* Irrigação Inteligente
* Uso de Bioinsumos

### 🐄 Pecuária Sustentável

* Criação de Bovinos
* Criação de Ovinos
* Criação de Caprinos
* Criação de Suínos
* Bem-estar animal
* Integração Lavoura-Pecuária-Floresta (ILPF)

### 🪨 Classificação de Solos

* Solo Argiloso
* Solo Arenoso
* Solo Humoso
* Solo Calcário

### ☁️ Monitoramento Agro

* Janela de plantio
* Alertas de pragas
* Indicadores de irrigação

### 🖼️ Galeria Interativa

* Filtro por categorias
* Imagens relacionadas à sustentabilidade agrícola

### 🎮 Minigame Educativo

* Simulador de tomada de decisões no campo
* Sistema de pontuação sustentável
* Cenários envolvendo:

  * Agricultura
  * Pecuária
  * Conservação do solo
  * Gestão de recursos naturais

### 📊 Calculadora de Impacto

Permite estimar:

* Redução de emissões de CO₂
* Economia de água
* Benefícios ambientais das práticas adotadas

### 💰 Simulador de Crédito Verde

* Simulação de financiamento sustentável
* Diferentes linhas de crédito
* Cálculo automático das parcelas

### 📧 Formulário de Contato

Canal para envio de mensagens e sugestões.
